Описание:
Симпотичный мини магазин на слайдерах с интересной анимацией,
подгрузка товаров из JSON (кнопка "SHOW MORE"),
выпадающие спойлер меню,
анимация добавления в корзину,
отзывчивая (привязка к курсору мыши) галерея-панно.
Интерфейс отзывчиво-адаптивный.
